CPSC 481 - Connect 4 AI Project
Team: Bryan Maus, Jericho Salonga, Logan Clampitt

FILES INCLUDED:
1. connect4_ai.py - The main playable game (Human vs AI). Run this to play.
2. evaluate_ai.py - The headless simulation script used to generate report data.
3. CPSC_481_Report.pdf - Final project report.

DEPENDENCIES:
- Python 3.x
- NumPy (pip install numpy)
- Pygame (pip install pygame)

HOW TO RUN:
1. Install dependencies: "pip install numpy pygame"
2. To Play: Run "python connect4_ai.py"
3. To View Stats: Run "python evaluate_ai.py" (Note: This runs 30,000 games and may take ~1-2 hours. Reduce 'num_games' in code for a quick test).
